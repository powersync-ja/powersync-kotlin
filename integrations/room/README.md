# Room integration for PowerSync

This package enables PowerSync for [Room](https://developer.android.com/training/data-storage/room)
databases, allowing you to define your queries in a type-safe way.
Watched Room queries automatically update when new data gets synced by PowerSync.

Note that this package is currently in _alpha_, and breaking changes are still expected.
It is tested however, and we encourage interested users to try it out!
